# AI Customer Service Agent Frontend

Vue 3 + Vite + Element Plus 前端，包含顾客侧 AI 客服、扫地机器人售前咨询、订单与退款、知识库页面，以及管理员工单处理页面。

## 主要页面

- **AI 客服**：支持多轮对话、历史会话搜索与软删除、售前需求摘要、商品卡片、详情抽屉和两款商品对比。
- **订单与退款**：查询订单、物流和退款申请状态。
- **知识库**：查看知识问答与来源信息；管理员可维护知识资料。
- **工单管理**：管理员可查看分类、优先级、情绪、AI 回复建议及历史案例来源。

## 本地开发

### 前置服务

默认开发代理会将 `/api` 转发到 `http://127.0.0.1:8080`，因此需要先启动：

1. MySQL（默认数据库为 `work_order`）。
2. Spring Boot 业务后端（8080）。
3. FastAPI AI 服务（8003），用于售前、知识问答、工单分析和回复建议。

完整启动说明见仓库根目录的 [README](../README.md)。

### 安装与启动

```powershell
npm install
npm run dev
```

浏览器访问 `http://127.0.0.1:5173`。

若后端端口不是 8080，请同步修改 `vite.config.js` 的 `/api` 代理目标，或通过 `VITE_APP_BASE_API` 指向完整 API 地址。

## 会话行为

- 点击“开始新对话”只创建本地草稿，不会产生空历史记录。
- 用户发送首条消息后，前端才调用后端创建会话并写入消息。
- 删除会话使用软删除，历史列表不再展示该会话，原消息记录保留在后端。

## 常用命令

```powershell
npm run dev
npm test
npm run build
```

`npm test` 运行页面和售前交互的静态接线测试；`npm run build` 生成生产静态资源。
